ALCHEMY POTION GENERATOR v1.0

The world's first Javascript powered D&D potion generator.
– by Adrien Jacquier Bret –

Inspired by "The Lemon King's Guide To Alchemy"


This generator isn't aimed to replace the original guide provided by its author. It is built to make potions in a more fast and convenient way, but too avoid any mistake or oversight, or even coding errors (yes it happens), please read carefully the complete guide before using this generator.
Also, be sure to compare results with your own calculation when something seems strange to you.
This software is open-source and its source code is available at this Github repository.
Please feel free to contribute and improve functionalities, and/or to give some feedback about things you'd like me to add or fix. Enjoy it !

If you'd like to have a map looking like this one used in the background, have a look at dharmablues's website !

Concentrates

If a player mixes ingredients in a retort or a flask and the ingredients share both positive and negative effects, the potion fails and the player learns nothing. Use of an alembic does not help in this situation because the player does not know what effects to filter out.

If, for some reason, an alchemist wants to make a low-potency potion, he or she may choose for the effect to be lessened by d6s, diminishing the potency by the result out of ten (ex: roll a 4, the potency is reduced by 4/10). The number of d6s must be chosen before rolling.
